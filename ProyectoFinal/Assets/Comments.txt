Como funciona la generacion de las dungeons..

de forma sensilla:
tenemos un sistema de celdas (un pasillo o una habitacion se considera una celda).
al iniciar la generacion lo primero que se instancia es una habitacion principal de donde donde hay 3 posibles lugares donde se puede posicionar una celda 
(a esto lo llamo temp portals).
se instancia una celda en la posicion  0,0,0..
se usa un physicscheck con las dimenciones de la celda en cada posicion de los temp portal para verificar que la celda no se encuentra superpuesta con otra.
si pasa el check la celda se queda en ese lugar.. si no lo pasa se mueve a otro temp portal.. si se genera una dungeon donde por x motivo es imposible intanciar
una nueva celda, a los 10 segundos se genera nuevamente.. 
despues de tener todas las celdas colocadas se reemplazan los temp portals que quedaron sin celda y se les coloca una pared.
en promedio tarda unos 6 segundos en generar una dungeon de 100 celdas (se puede modificar la cantidad)..

deje todo comentado el script de DungeonManager espero que se entienda porque soy bastante malo explicando, y mas malo comentando scripts (castro franco)...
si queres ver las celdas estan en assets/prefabs/dungeongen.. ahi hay una carpeta de rooms y de hallways (las celdas).. cada una tiene un script "cell.cs"
que lo unico que hace es serializar los temp portals y asegurarse de que tenga un box collider...

a futuro me gustaria mejorar el script para tener en cuenta las formas irregulares de las celdas, para optimizar la generacion y disminuir bugs.

    Los enemigos se instancian una vez terminado de generar la dungeon, en distintas celdas, en puntos especificos de spawns,
los cuales cada uno tienen una probabilidad de generar un enemigo, tiene un array de Game Objects para los enemigos que 
pueden spawnear, y un array para la probabilidad de spawn cada enemigo. Al instanciarse cada enemigo, se mueven como hijos
de Enemy Container.
Luego de un tiempo se asignan todos los hijos de Enemy Container a Enemy Count en enemies left, cuando se mate a cierta cantidad
se sumara un punto y se volvera a generar la dungeon
